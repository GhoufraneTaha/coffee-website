# Coffee à Gout - Website

## Overview
**Coffee à Gout** is a modern and visually appealing website for a coffee shop. The website is designed to showcase the coffee brand's premium offerings, cozy atmosphere, and community engagement. It provides users with an intuitive interface to explore the menu, learn about the coffee shop, connect with the community, and place orders online.  

The project demonstrates front-end web development skills using **HTML, CSS, and responsive design principles**, along with attention to user experience and visual aesthetics.  

---

## Features
- **Home Page:**  
  - Hero section with high-quality coffee shop images.  
  - “Explore Menu” button for easy navigation.  
  - “Why Choose Us?” section highlighting premium coffee, cozy space, and ethical practices.  
  - Featured coffee drinks with visually appealing cards and hover effects.

- **Menu Page:**  
  - Showcases the full coffee menu with images and descriptions.  
  - “Order Now” button for quick access to online orders.  
  - Responsive design for different screen sizes.

- **Community Page:**  
  - Highlights events, social engagement, or coffee-related activities (optional feature).  

- **About Page:**  
  - Shows the coffee shop story, process, and ethical approach.  

- **Order Page:**  
  - Allows users to select coffee size, toppings, and place items in a cart.  
  - Dynamic cart preview with item removal functionality.

- **Contact Page:**  
  - Contact form for inquiries or feedback.  
  - Responsive and accessible layout.

---

## Technologies Used
- HTML5  
- CSS3 (with gradients, shadows, and hover effects)  
- Responsive web design (media queries for mobile, tablet, desktop)  
- Flexbox for layout and card designs
